#모듈:코드를 잘모아서 기능하나를 모아놓아 구현할수있는 파일(코드들어있는파일)
(이코드가 모여서 어떤기능을 구현함 )

animal1(package)이라는 파일을 만들고 그안에 cat,dog(modules)라는 파이썬파일을 만든다
그곳에서 class를 써서 Dog,Cat을 정의하자
그다음 __init__이라는 파이썬 파일에서 
from .cat import Cat        # .의 의미는 "Cat.py"에서 Cat이라는 클래스를 가져온다
from .dog import Dog        # .의 의미는 "Dog.py"에서 Dog이라는 클래스를 가져온다
을 만들자!!

그다음 다른 라이브러리 파일에서 

from animal1 import dog      ##animal이라는 패키지에서 dog라는 모듈 갖고와줘
from animal1 import cat
d=dog.Dog()
d.hi()

c=cat.Cat()
c.hi()


from animal1 import * #animal 패키지에서 모든모듈 다 갖고와
d=dog.Dog()
d.hi()

c=cat.Cat()
c.hi()
